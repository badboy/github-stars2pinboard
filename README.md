# github-stars2pinboard

Save all [starred][] github repos to [Pinboard][] using [octokit][] and the [pinboard gem][].

## Usage

    ./gh-stars2pinboard.rb [api_token]

Get your `api_token` on <https://pinboard.in/settings/password>
